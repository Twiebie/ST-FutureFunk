# Future Funk #
An easy to read dark color scheme for Sublime Text 2 and 3.

## Installation ##
1. Clone the repo to your Packages folder.
2. Activate the color scheme via Preferences -> Color Scheme -> ST-FutureFunk -> Future Funk
